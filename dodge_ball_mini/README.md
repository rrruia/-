# 小球躲避游戏 Dodge Ball Mini

一个轻量级的 HTML5 Canvas 休闲小游戏。玩家在屏幕底部控制挡板水平移动，躲避从顶部不断下落的敌方小球。坚持更久即可获得更高分数。

## 主要特性
- 纯前端实现：HTML + CSS + JavaScript，无后端依赖
- 多种敌人行为：直落、弹跳（水平来回）、加速（下落速度递增）
- 难度递增：随时间加快生成频率与下落速度，并可能同时生成多个敌球
- 交互完整：暂停/继续（按钮、Esc/空格），Game Over 面板，居中继续按钮显示当前分数
- 说明面板：点击“说明”查看玩法与操作，查看说明时自动暂停
- 移动端适配：在画布上“滑动”即可控制挡板（禁用点击跳位），防滚动、防选中

## 操作说明
- 键盘：左/右方向键 或 A/D 进行水平移动
- 暂停：Esc 或 空格（说明面板打开时禁用快捷暂停）
- 移动端：在画布上“滑动”控制挡板，仅滑动有效，轻触不会直接跳位

## 难度设计
- 每隔一段时间提升难度：缩短生成间隔、提高下落基础速度
- 动态敌型概率：随生存时间提高，弹跳/加速型敌人出现更频繁
- 同时生成：在中后期一次生成 2–3 个敌球，提升压迫感

## 运行与预览
- 本地运行：在项目目录下执行
  - `python -m http.server 5500`
  - 浏览器打开 `http://localhost:5500/ball_drop.html`
- 局域网分享：其他设备可通过你的电脑局域网 IP 访问 `http://<你的IPv4>:5500/ball_drop.html`
- GitHub Pages 部署：
  - 将页面文件放在仓库根目录或 `docs/` 目录（需包含 `ball_drop.html` 与相关静态资源）
  - 仓库 Settings → Pages：选择 `Branch` 与 `Folder`（`/` 或 `/docs`）
  - 访问生成的公开地址即可游玩

## 文件结构
- `ball_drop.html`：页面入口
- `styles.css`：页面与覆盖层样式（含移动端触控配置）
- `src/game.js`：游戏核心逻辑（玩家、敌人、碰撞、难度、UI交互、输入事件）

## 可扩展性
- 敌人类型：曲线运动、追踪型、分裂型
- 道具系统：护盾、减速、清屏、短暂无敌
- 视觉与音效：粒子特效、碰撞音效、主题皮肤
- 排行榜：本地最高分或连网排行

## 设计约束与兼容性
- 鼠标控制已禁用：桌面端仅支持键盘；移动端仅支持滑动
- 在 Edge 浏览器中已提升 UI 层级与点击可用性（设置 `z-index` 与 `pointer-events`）
- 说明面板打开时，Esc/空格暂停快捷键被屏蔽，关闭说明后恢复
